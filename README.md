OpWeb
=====
a easy and Real Time develop game/web frame

[![Bitdeli Badge](https://d2weczhvl823v0.cloudfront.net/icesun963/opweb/trend.png)](https://bitdeli.com/free "Bitdeli Badge")

### Try it now!!

Download and unzip to a folder

cd OpWeb

node AppLauncher.js

Open On Browser:

http://localhost:8000

(Don't work in IE)
